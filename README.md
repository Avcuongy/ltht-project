# How to contribute

1. Clone the repository: `git clone https://github.com/Avcuongy/ltht-project.git`
2. Restore Packages and Build:
    
    - Option 1: Using Visual Studio Community: Open `ltht-project.slnx`. The IDE should automatically restore NuGet packages. If not, right-click on `Solution Explorer` and select Restore NuGet Packages.

    - Option 2: Using CLI `dotnet restore` -> `dotnet build`
